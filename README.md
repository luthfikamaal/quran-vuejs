# Quran - Vue.js

## Installation

Run this commands

```
git clone https://github.com/luthfikamaal/quran-vuejs.git
cd quran-vuejs
npm install
npm run dev
```

## Tool

|               |             |
| ------------- | ----------- |
| JS Framework  | Vue 3       |
| CSS Framework | TailwindCSS |

## Data Source

- Quran API : [https://api.alquran.cloud/v1/surah/1](https://api.alquran.cloud/v1/surah/1)
